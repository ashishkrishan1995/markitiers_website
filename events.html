Events Mosaic

Events Mosaic, an initiative by markitiers, an event discovery and logistic application with a vision to build an events community globally.

It aims to cater to the needs of all the parties involved in making an event successful and envisons to provide oppurtunities to brands, startups, NGOs revolutionising the way events are brought together. Being a youth led community, it provides brands an opportunity to interact with the youth using various enagement activities and events.

For any collobations contact - eventmosaic@markitiers.com
